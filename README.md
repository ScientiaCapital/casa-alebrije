# Casa Alebrije

Boutique hotel website for Casa Alebrije - 7 unique suites inspired by Mexican spirit animals, located in Coyoacán, Mexico City (5 minutes from Frida Kahlo Museum).

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4
- i18next (EN/ES)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- 7 themed suites (Armadillo, Jaguar, Hummingbird, Octopus, Owl, Butterfly, Quetzal)
- Bilingual support (English/Spanish)
- Responsive design
- Location with Google Maps

## Project Structure

```
src/
├── components/
│   ├── layout/      # Header, Footer, LanguageToggle
│   ├── sections/    # Hero, RoomsPreview, About, Location, Testimonials
│   ├── rooms/       # RoomCard
│   └── ui/          # Button, Card
├── data/            # rooms.ts, testimonials.ts
└── i18n/            # Translations
```
